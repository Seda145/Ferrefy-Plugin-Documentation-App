## About this app
---

This app provides documentation for my Unreal Engine plugins.

Download the Git repo / zip to a folder on your PC, then double click the /App/app.htm file to open it in your PC browser. It should load just like a website. Done!

The app works 100% offline.  



The technical design differs quite a bit from my other offline apps. Originally I had built a private "builder" app that would build and export my Ferrefy website. The exported website (that I hosted to the public) contained only the super lightweight files required for a running app, not for building an app. There was no back end running online (again lightweight, also secure), since none was required.

Some time ago I took that Ferrefy website offline and decided to just make a part of the "builder" program a public Git repo. Instead of exporting a website, it now builds the Ferrefy website to work locally on your PC, then runs it. So, that explains the amount of js files and overall technical design.

Of course the app runs exactly like the Ferrefy website did, and that technical info was added just for the curious.

You can see some of my more recent app development in my apps "Ferret Tale" and "Heartbeat".
